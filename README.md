## Run Locally   return

Clone the project

``` console
git clone https://github.com/Harrytimbog/WOLF_MANAGER.git
```

Go to the project directory

``` console
cd WOLF_MANAGER
```

Install dependencies

``` console
npm install
```

Install gems

``` console
bundle install
```

Start Server

``` console
rails s
```
