## Run Locally   

Clone the project

``` console
git clone https://github.com/Harrytimbog/WOLF_MANAGER.git
```

Go to the project directory

``` console
cd WOLF_MANAGER
```

Install dependencies

``` console
npm install
```

Install gems

``` console
bundle install
```

Create, Migrate and Seed Database

``` console
rails db:create db:migrate db:seed
```

Start Server

``` console
rails s
```
